#+title: eat-windows-pty — ConPTY support for emacs-eat on native Windows
#+author: Fujisawa Electric Management Office

#+begin_quote
Make =M-x eat= work on native Windows Emacs.
#+end_quote

* What is this?

Native Windows Emacs has no PTY support. Upstream [[https://github.com/akib/emacs-eat][eat]] wraps commands
with =/usr/bin/env sh -c ...= which fails on Windows.

This package spawns =conhost.exe --headless --feature pty= as a
ConPTY wrapper, giving eat a proper pseudo-console on Windows.

* How it works

#+begin_example
Emacs (pipe-mode subprocess)
  └─ conhost.exe --headless --feature pty PROGRAM ARGS...
       └─ ConPTY pseudo-console
            └─ Your shell / program
#+end_example

* Requirements

- Emacs 29.1+ (native Windows build)
- [[https://github.com/akib/emacs-eat][eat]] 0.9+
- Windows 10 1809+ (ConPTY support)

* Installation

** async-installer

#+begin_src emacs-lisp
(async-installer-github-add "https://github.com/zawatton21/eat-windows-pty.git"
                            :branch "master"
                            :main "eat-windows-pty.el")
#+end_src

** straight.el

#+begin_src emacs-lisp
(straight-use-package
 '(eat-windows-pty :type git :host github :repo "zawatton21/eat-windows-pty"))
#+end_src

** elpaca

#+begin_src emacs-lisp
(elpaca (eat-windows-pty :host github :repo "zawatton21/eat-windows-pty"))
#+end_src

** use-package (with straight or elpaca)

#+begin_src emacs-lisp
;; with straight
(use-package eat-windows-pty
  :straight (eat-windows-pty :type git :host github :repo "zawatton21/eat-windows-pty"))

;; with elpaca
(use-package eat-windows-pty
  :ensure (:host github :repo "zawatton21/eat-windows-pty"))
#+end_src

** Manual

Clone this repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/eat-windows-pty")
(require 'eat-windows-pty)
#+end_src

* Usage

#+begin_src emacs-lisp
(with-eval-after-load 'eat
  (when (eq system-type 'windows-nt)
    (require 'eat-windows-pty)))
#+end_src

That's it. =M-x eat= now works on Windows.

* Design

Drop-in override, not a fork. Redefines two functions after eat loads:

- =eat-exec= — routes through =conhost.exe= instead of =/usr/bin/env sh=
- =eat--1= — passes program directly instead of shell-wrapping

Based on [[https://github.com/akib/emacs-eat/pull/126][PR #126]] by Artem Khramov, never merged upstream.
